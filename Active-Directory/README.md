# Enterprise Active Directory Infrastructure

## Overview

This project documents the design, deployment, administration, validation, and operational management of a Microsoft Windows Server 2022 Active Directory environment developed as part of the DouglasTech Enterprise Infrastructure Framework.

The implementation demonstrates enterprise identity management, centralized authentication, DNS integration, Group Policy administration, Organizational Unit (OU) design, security group management, Windows 11 domain integration, and Active Directory troubleshooting.

The project follows a structured engineering methodology emphasizing documentation, repeatability, validation, change management, and operational consistency.

---

## Business Objectives

- Deploy Microsoft Windows Server 2022 as an Enterprise Domain Controller.
- Configure Active Directory Domain Services (AD DS).
- Implement enterprise DNS services.
- Design and administer Organizational Units (OUs).
- Create and manage domain users and security groups.
- Configure and administer Group Policy Objects (GPOs).
- Integrate Windows 11 Enterprise into the Active Directory domain.
- Validate authentication, directory services, and DNS functionality.
- Troubleshoot Active Directory, DNS, networking, and trust relationship issues.
- Document the complete deployment using enterprise engineering standards.
